    Colors

    The Primary Color Palette has a HEX value of #A4E5A3 and RGB value of 164 229 163.
    The Secondary Palette uses the shade color rule. THe HEX values:76A576, 496548, A4E5A, ADF2AC, 92CB91.
    The Secondary Palette uses RGB values:118 165 118, 73 101 72, 164 229 163, 173 242 172, 146 203 145.

     

    Typography

    The following fonts are used for this site:

    • Headings:
      font-family: 'Crete Round', serif;
    • Body Copy:
      font-family: 'Montserrat', sans-serif;

    Headings

    All HTML headings, <h1> through <h4>, are available. .h1 through .h4 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.
